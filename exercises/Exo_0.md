# Exo 0: Introduction

- Create your own repository by ["Use this template"](https://github.com/new?template_name=exoHelloWorld&template_owner=pSciComp).
- If you want to have exemplary solutions for the exercises included, be sure to **select "Include all branches"** when creating a new repository based on this template.
  
  You are not forced to use GitHub here, can also simply clone this repository locally, create an empty repository on the git remote service of your choice (e.g. [codeberg.org](https://codeberg.org)) and update the url of origin in your cloned repo:

  ```bash
  git remote set-url origin git@<new-remote>:<username>/<reponame>.git
  ```
- Once you've created your own copy of the repository, you're ready to go.
- Head over to [Structure Exercise 1](./structure/Exo_1.md) to get started.
- If you have questions you might check the exemplary solutions available under the corresponding branches, i.e. solutions to _Strucuter Exercise 1_ can be found in branch _struct_exo_1_.
  Alternatively, you can also [open an issue](https://github.com/pSciComp/exoHelloWorld/issues/new) on the template repository or checkout the [discussions](https://github.com/pSciComp/exoHelloWorld/discussions).
- **Invite us to your repository:**
